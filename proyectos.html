Todos mis proyectos

API administración de comercio

Este proyecto es una API RESTful desarrollada con Spring Boot que permite gestionar clientes, ventas y productos dentro de un sistema de administración de comercios. Ideal para integrarse con plataformas web o móviles.

App. login de usuarios

Esta aplicación de escritorio desarrollada en Java con Swing permite la administración de usuarios con sistema de login y almacenamiento de datos en MySQL. Ideal para sistemas internos de gestión en pequeñas y medianas empresas.

App. peluquería de Mascotas

Esta aplicación de escritorio desarrollada en Java con Swing permite gestionar una peluqueria de mascotas mediante operaciones CRUD de dueños, mascotas y consultas, utilizando MySQL como base de datos.

Tienda Zister indumentaria

Esta tienda online fue desarrollada con JavaScript, HTML, CSS y Bootstrap, y permite explorar, filtrar y gestionar productos de ropa de manera dinámica y responsiva. Se presenta como una solución efectiva para emprendimientos digitales en el sector indumentaria.

Landing page venta GPUs

Esta landing page está construida con HTML, CSS y JavaScript, y presenta de forma atractiva una selección de GPUs para venta online. Tiene un estilo visual inspirado en los videojuegos, pensado para atraer a entusiastas del hardware y la vida Gamer!

Calculadora web

Calculadora web creada con HTML, CSS y JavaScript. Permite sumar, restar, multiplicar y dividir con una interfaz simple e intuitiva. Es un proyecto que combina estructura, estilo y lógica en una aplicación funcional y fácil de usar.

Web encuesta GamerStore

Página web gamer con formulario de satisfacción creada con HTML, CSS y JavaScript. Permite a los usuarios calificar la experiencia, con diseño atractivo y dinámico. Ideal para recopilar opiniones y mejorar el sitio según los comentarios recibidos.
